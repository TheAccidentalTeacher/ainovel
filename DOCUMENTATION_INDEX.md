## üöÄ Getting Started

### Installation & Setup
1. **Prerequisites**: Python 3.12+, Node 20+, MongoDB, API keys (Anthropic, OpenAI, Tavily)
2. **Quick Start**: Follow [README.md Setup Instructions](README.md#local-development)
3. **Environment Configuration**: Copy `.env.example` to `.env` and populate keys
4. **Database Setup**: MongoDB collections auto-created on first run

### First Novel Workflow
1. Create a new project via Premise Builder
2. Generate Story Bible (character/setting extraction)
3. Generate Outline (structured chapter planning)
4. Generate chapters (bulk or individual)
5. Export manuscript
